ffmpge的基本使用

# 下载

ffmpeg官网 (opens new window)

  1. 打开https://ffmpeg.org/点击Download
  2. 选择window - Windows builds from gyan.dev
  3. 选择 https://www.gyan.dev/ffmpeg/builds/ffmpeg-git-github
  4. 选择 ffmpeg-2022-03-31-git-e301a24fa1-full_build.zip下载
  5. 解压-重命名-移动到D: D:\ffmpeg
// 在下载zip文件夹下打开cmd
WinRAR X ffmpeg-2022-03-31-git-e301a24fa1-full_build.zip
xcopy /s /e /i /y ffmpeg-2022-03-31-git-e301a24fa1-full_build D:\ffmpeg\ 
setx "path" "D:\ffmpeg\bin;%path%" /m

exit
// 重启cmd窗口查看版本号
ffmpeg -version

// 一个媒体文件里边有多个音视频流，各个流的时长也未必一样，一般播放器会以video stream的时长作为播放时长，另外，并不是所有格式的视频，会在音视频流级别保存了时长，这时可能会拿不到时长
// 解码后文件播放时长（get duration by decoding）

// 容器时长（container duration)
// 特点：慢、准确
ffprobe -v error -show_entries format=duration -of default=noprint_wrappers=1:nokey=1 -i input.mp4

# 常用命令行参数

-c：指定编码器
-c copy：直接复制，不经过重新编码（这样比较快）
-c:v：指定视频编码器
-c:a：指定音频编码器
-i：指定输入文件
-an：去除音频流
-vn： 去除视频流
-preset：指定输出的视频质量，会影响文件的生成速度，有以下几个可用的值
	    ultrafast, superfast, veryfast, faster, fast, medium, slow, slower, veryslow。
-y：不经过确认，输出时直接覆盖同名文件。

# 压缩音频

在cmd中输入命令并运行：
ffmpeg -i "Electron超详细实战开发教程.wav" -b:a 64k -acodec mp3 -ar 44100 -ac 1 "Electron超详细实战开发教程.mp3"
命令参数解释：
-i "1.wav"表示输入，后面的文件名必须要用双引号；
-b:a 64k表示音频的码率为64kb/s，即8kB/s；
-acodec mp3表示设定声音编解码器为mp3；
-ar 44100表示设定采样率为44100；
"1.mp3"表示输出保存的文件名。

ffmpeg -i "Electron超详细实战开发教程.wav" -ar 500 "out.wav"

# 挂载字幕


# ffmpeg -i input.mp4 -i input.srt -c:s copy -c:v copy -c:a copy out.mp4
# 添加硬字幕，ffmpeg最终都会将字幕格式先转换成ass字幕流再将字幕嵌入到视频帧中，这个过程需要重新编解码，所以速度比较慢。
ffmpeg -i input.mp4 -vf subtitles=input.srt output.mp4
ffmpeg -i input.mp4 -vf subtitles=input.ass output.mp4
# 添加软字幕，添加软字幕的原理和流程就跟给视频添加音频一样，这个过程不需要重新编解码，所以速度非常快。
ffmpeg -i input.mp4 -i input.srt -c copy output.mp4
ffmpeg -i input.mp4 -i input.ass -c copy output.mkv

ffmpeg -i input.mkv -i input.ass -c copy output.mkv

给mkv文件添加srt格式的软字幕：
ffmpeg -i input.mkv -i input.srt -c copy -c:s srt output.mkv
给mp4文件添加srt格式的软字幕(mkv格式本身支持字幕轨道，但mp4需要使用mov_text）
ffmpeg -i input.mp4 -i input.srt -c copy -c:s mov_text output.mp4
给mkv文件添加ass软字幕(mp4不支持ass编码格式的字幕流,mkv支持）
ffmpeg -i input.mp4 -i input.ass -c copy -c:s ass output.mkv



ffmpeg -i "1.最新Electron课程介绍.mp4" -i "1.最新Electron课程介绍.srt" -c copy "output.mp4"
ffmpeg -i "1.最新Electron课程介绍.mp4" -i "1.最新Electron课程介绍.ass" -c copy "output.mp4"
ffmpeg -i 1.最新Electron课程介绍.mp4 -i 1.最新Electron课程介绍.srt -c:s copy -c:v copy -c:a copy out.mp4

ffmpeg -i 1.最新Electron课程介绍.mp4 -i 1.最新Electron课程介绍.ass -c:s copy -c:v copy -c:a copy out.mp4
